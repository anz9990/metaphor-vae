# metaphor-vae

Interpolating Metaphors with Variational Autoencoders

# Dependencies

* Python 3.6
* Pytorch 0.4
* Fastai 0.7

# Installation

1. Install the dependencies
2. Add `fastai/old/` to `PYTHONPATH`
